自身のスキルを掲載する
HTML
・HTML Living Standardの仕様に基づいたマークアップ
・適切な見出しの設定（アウトラインを理解している）
・picture要素を使い、画面解像度や画面幅、画像形式等に基づいて、開発者が任意の画像を出し分ける「アートディレクション」が可能。
・フォーム関連の新しい属性やautocomplete属性などを使う事で、今までJavaScript等で行う事が一般的だった入力補助や入力チェックなどの機能を簡単に導入でき、短い作業時間でユーザーにとってもEFOを考慮した、解り易いフォームを作成する事が出来ます。
CSS
・cssでネストを使った記述
・cssの拡張言語であるsassを使ったコーディング
・「can I use」等を利用して各ブラウザの対応状況を確認
   can I useでブラウザの対応状況を調べて、使えないデバイスの対応をする
・カスタムプロパティ（変数）を使用し、コードのメンテナンス性を高める
Javascript
・date関数を使って時間や日時の表示
・変数や関数（forやif）などプログラム全般で必要とされる基本的な考え方を習得
jQuery
・Ajaxを使ったファイル操作
・酔うようなプラグインの実装
・css3のtransformと連動させたアニメーション機能と連動（ハイブリッドアニメーション）
 positionではなくてtransform、transitionで滑らかに動かす（positionはカクつく）
PHP
・postとgetでのデータ送信
・ユーザビリティの優れたフォーム作成
 autocomplete機能を使って住所や電話番号などを素早く表示
・xammp/mampを使った仮想サーバーの構築
Wordpress
・オリジナルテーマの作成（html、css、画像）
・カスタムフィールドを使ったデータ取得
・外部記事データの読み込み
